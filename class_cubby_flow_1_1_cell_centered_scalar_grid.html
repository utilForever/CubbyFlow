Detailed Description

template<size_t N>
class CubbyFlow::CellCenteredScalarGrid< N >

N-D Cell-centered scalar grid structure.

This class represents N-D cell-centered scalar grid which extends ScalarGrid. As its name suggests, the class defines the data point at the center of a grid cell. Thus, the dimension of data points are equal to the dimension of the cells.
